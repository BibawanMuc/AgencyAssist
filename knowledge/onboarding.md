# Onboarding Knowledge Base

Diese Datei enthält das Onboarding-Wissen für den Chatbot.

## Anleitung

1. Öffnen Sie Docmost und navigieren Sie zum Onboarding-Guide
2. Klicken Sie auf das Menü (drei Punkte) → "Export" → "Markdown"
3. Speichern Sie die heruntergeladene Datei
4. Ersetzen Sie diese Datei mit dem exportierten Inhalt

**Wichtig:** Der Chatbot kann erst auf das Wissen zugreifen, wenn diese Datei mit dem echten Onboarding-Inhalt gefüllt ist.
